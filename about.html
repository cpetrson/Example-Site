A number of years back, I had picked up enough HTML and CSS to make a website that sat locally on my laptop, but I needed a host and a server in order to get my feet wet with PHP. (I discovered Local later!) After doing some research, I purchased space on a2hosting.com along with a domain name, and I ran their Softaculous installer to set up WordPress. Learning my way around WordPress and cranking out first a child-theme and then a full-fledged theme was a great introduction to theme developement. From discovering the difference between WordPress.com and WordPress.org, to implementing Custom Post Types, this project was a fantastic way to work with WordPress.
