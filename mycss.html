CSS - Cascading Style Sheets

Stylish and Responsive Design

Fall 2024 - present

Recently I picked up Brad Schiff's Udemy tutorial on HTML & CSS (see below). This tutorial covers so many fun things, including Bootstrap, Sass, and pushing a website up onto GitHub's hosting for free. There is also a thorough example of mobile-first classes with standard breakpoints for responsive design. Here are some highlights from that tutorial.

One photo & palette. Different navigational styles.

As I stepped through Schiff's tutorial, I wanted to try the concepts out in my own workspace. Taking an original photo of apple blossoms as a starting point, I used it as the base for working with the different styles the tutorial was demonstrating. A palette 'grew' from the photo, as did the sample pages as I experimented with different style elements.

Although I had been working with HTML and CSS in various projects, I wanted to see what this tutorial could show me that was new. It is both a fantastic place to start for anyone interested in working with front-end development as well as a great way to branch into important topics such as grid systems, Bootstrap, and Sass. While it doesn't require programming know-how, Schiff sets the stage for Javascript and explains why it is the third member of the 'happy family' along with HTML and CSS.
