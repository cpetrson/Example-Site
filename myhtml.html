HTML - Hypertext Markup Language

Building block(s) of the Web

2019 - 2020

In the beginning... (it actually wasn't the beginning, but we'll start there) ...I wanted to understand HTML elements. Where to start with HTML? By building a webpage! Within a few clicks, I discovered the tutorial pages at https://www.w3schools.com/html and was on my way, stepping through their guide and meeting element after element, attributes and functions enough to make any web explorer happy (almost). Making a local site of interworking pages to test out elements as I went, the web pages below were the result of my year or so of exploring the tutorial.

Project:

This project started as a free-form place to test out the html elements as I learned about them, but it soon expanded into more: a navigation header, css, javascript... and when I started building pages with forms, I wanted to be able to handle the HTTP header key/value pairs and global variables and get my hands on the values I could pass from the form. But this required PHP and server-side access, and a database or some form of persistance if I wanted to keep those values, and most of all the ability to work with PHP... which led me into the next project: WordPress.
