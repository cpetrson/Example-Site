A LAMP Stack from the Ground Up

Putting the foundation under a WordPress website.

Fall 2023 - Spring 2024

From installing Linux (Ubuntuu LTS 22), Apache, MySql, PHP, ...and WordPress, to configuring Apache for serving up multiple sites accessible from our LAN, this was a great project to get familiar with the components that WordPress sits on top of. Some highlights:

This is our LAMP server project, starting with Ubuntu.

Project:

In chatting about web servers, WordPress, and sites, the idea of building our own LAMP Stack on some repurposed equipment took shape. This old equipment was repurposed by installing Ubuntuu LTS 22 Server and Desktop, then the remaining LAMP stack components on the server: Apache, MySql, and PHP. WordPress went on top of that.

Did I mention that it's not a good idea to install Maria DB over MySql?
